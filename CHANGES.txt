Version 9.0.32
--------------

This is a significant new release of the connector with a new synchronizer
and a number of improvements to the management of content items.

[#371] Gradebook name-clash possible when renaming schedules:
	This issue has been fixed for content items
[#326] Make Schedule Unavailable
	Added checkbox to enable standard content availability flag to be set
[#321] Save schedule restricted times in Blackboard content item
	Content items now synchronize standard content availability times with schedule
[#209] Use tree-views for Assessment list browsing
	Forms now use a tree-like view for browsing list of available assessments
	This may also resolve overly restricted permission issues
[#216] Manual Synchronization Option for Large Courses
[#325] Improve Synchronization
	All new 'just-in-time' synchronization process should work for all sizes of course
	Course members are now synchronized *individually* when they access the connector
	so students have access to assessments as soon as they enroll (no need for the
	instructor to force the synchronization in the control panel).  Manual synchronization
	is still possible if desired.
[#224] Automatic Creation/Linking to Assessment and Topic folders
	Experimental support for synchronizing assessment folders and permissions
[#361] Limit use of the connector to specified courses
	Group synchronization can now optionally be disabled allowing Perception-side control
	of which courses are synchronized if desired.
[#358] Syncrhonize Email Address with Perception
	Participant email addresses now synched to Perception

Minor bug fixes:
[#316] Users without a role in the Blackboard course get an error message
[#311] Schedule name appears twice in the breadcrumb trail
[#334] Connector setting page does not remember https setting

Version 9.0.8
-------------

Includes fixes for:

[#212] Compatibility with adaptive release

Instructors can now hide the list of schedules from the student view, the
schedules can still be accessed by the student using a schedule specific URL
that can be copied from the instructor's view and pasted into content items.

Note that schedules are NOT securely hidden, determined students can view their
own schedule if they know the schedule name.  Permission to take the assessment
is of course controlled by the schedule settings and is not affected.

[#219] Reporting of percentage scores

Added a three way option to support percentage, point scores or no gradebook
entry at all.  By default, percentage is chosen.

Note that a schedule may be retrospectively converted to percentage format by
changing the "Points Possible" in the grade centre to 100.  Existing scores are
NOT converted so do not change this setting unless it is set to 0, indicating
that no results have been received yet.

[#208] Moved results view to a separate page
